# nmon
https://sourceforge.net/projects/nmon/files/

* CPU 使用率
* 内存使用情况
* 内核统计信息和运行队列信息
* 磁盘 I/O 速度、传输和读/写比率
* 文件系统中的可用空间
* 磁盘适配器
* 网络 I/O 速度、传输和读/写比率
* 页面空间和页面速度
* CPU 和 AIX 规范
* 消耗资源最多的进程
* IBM HTTP Web 缓存
* 用户自定义的磁盘组
* 计算机详细信息和资源
* 异步 I/O，仅适用于 AIX
* 工作负载管理器 (WLM)，仅适用于 AIX
* IBM TotalStorage® Enterprise Storage Server® (ESS) 磁盘，仅适用于 AIX
* 网络文件系统 (NFS)
